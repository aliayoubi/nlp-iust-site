نتايج آزمايشات بر روی نسخه 0.1 پيکرة وابستگی زبان فارسی

در اين صفحه تنظيمات و نتايج آزمايشات انجام شده در پايان‌نامه "ساز و کاری برای کشف تأثير ويژگی‌های مختلف ساخت‌واژی و صرفی بر روی تجزية وابستگی زبان فارسی" بر روی دو تجزيه‌گر MaltParser و MSTParser ارائه شده است.

انتخاب الگوريتم مناسب در MaltParser

الگوريتم‌های مختلف موجود در MaltParser امتحان شدند که نتايج آن در جدول 1 ارائه شده است. در اين ميان الگوريتم کاوينگتون غيرافکنشی بهترين عملکرد را در تجزية وابستگی زبان فارسی داراست. در ادامة آزمايشات از اين الگوريتم استفاده شده است. کلية آزمايشات با تنظيمات پيش‌فرض و با استفاده از دستور زير اجرا شده است. 

java -jar maltParser.jar -a (algorithm-name)

انتخاب الگوريتم مناسب در MSTParser

در ابزار MSTParser دو الگوريتم افکنشی (آيزنر) و غيرافکنشی (چو-ليو-ادموندز) در دو حالت مرتبه اول و دوم وجود دارد. نتايج حاصل از اجرای اين الگوريتم‌ها بر روی زبان فارسی در جدول 2 آمده است. الگوريتم مرتبه دوم غيرافکنشی بهترين عملکرد را داشته که در ادامه از اين الگوريتم استفاده شده است.

بهينه‌سازی MaltParser

به منظور بهينه‌سازی از ابزار MaltOptimizer استفاده شده است. 

						// Run phase 1
						java -jar MaltOptimizer.jar -p 1 -m maltParser.jar -c input.conll
						
						// Run phase 2
						java -jar MaltOptimizer.jar -p 2 -m maltParser.jar -c input.conll
						
						// Run phase 3
