Result of experiments on the persian dependendency treebank (version 0.1)

In this page, experimental settings and results which had been applied on the MaltParser and MSTParser in the MSc. Thesis "A mechanism for exploring of the effect of different morphologic and morphosyntactic features on Persian dependency parsing" is presented.

Selecting best algorithm between algorithms of MaltParser for Persian language

Table 1 shows results of aplying all different algorithms which are existed in MaltParser. It shows that non-projective covington algorithm leads to best accuracy. In all algorithms we use default settings by invoking below command: 

java -jar maltParser.jar -a (algorithm-name)

Selecting best algorithm between algorithms of MSTParser for Persian language

In MSTParser, there are two algorithms for Projective (Eisner) and Non-Projective (Chu-Liu-Edmonds) that can be used in first and second order mode. Results of applying this algorithms on the persian language is presented in Table 2. It shows that non-projective second order algorithm has best accuracy.

Table 2. Results of applying different algorithms in MSTParser on the Persian Language
Order Type Accuracy
First Projective 82.94
Non-Projective 83.59
Second Projective 83.74
Non-Projective 84.59
