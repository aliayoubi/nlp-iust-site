Dependency Parsing Toolbox

There are various tools for dependency parsing, But until now, an integrated environment with different algorithms and methods of dependency paring hadn't been designed. In this toolbox, different existing algorithms and their related requirements are collected or developed in one place. All of existing parsers is ran in command line mode and any of them have its specific parameters and it forced you to read their documents when you want to work with them. In this toolbox we has designed an user interface on top of existing tools that has simplified usage of them. You can read more about each parts here:

  • Type Converter
  • Reading Corpus
  • Create Dependency Graph
  • Projectivizing
  • Optimizing
  • Training
  • Parsing
  • Evaluating
  • Hybrid Methods
